Part 1: What is a Pi?

In brief, The Raspberry Pi is a:

  • tiny, full-featured computer.
  • platform for creating with code and electronics.
  • "System on a Chip" ideal for embedded systems.

To build an art installation, measure rainfall, make a light-up dress – none of those things sound like computer science but they’re all things that computers can help you do – that’s the message we’re really trying to get across.

- Eben Upton

Part 2: Kit Components

Raspberry Pi 3A+

The heart of it all. Smaller than the Pi 4, it has a 64-bit processor, wi-fi and Bluetooth.

And, of course, HDMI out.

10.1 inch IPS Display

Designed for the Pi, this includes an HDMI connector, speakers, and a little stand.

It's also a touch screen!

SD Card

This will contain the operating system, applications, and all the media on our frames.

5V 2.5A Power Supply

Just what it sounds like.

The Raspberry Pi 3A+ can also be powered by a micro USB cable connected to a computer or USB wall adapter.
+ optional battery
+ optional frame and mat dimentions

Part 3: OS Install

Download the Imager

www.raspberrypi.com/
software/

Choose an OS

Select:
Raspberry Pi OS (other)

Then select:
Raspberry Pi OS Lite (Legacy)

Select SD Card

Look for your SD card in the pop-up window. It may have a generic name.

Name your Pi & create a user

Click on the gear icon. Create a hostname, a username, and add WiFi and locale information.

Write the OS

Hit the write button. Once complete, eject the SD card and insert it into your Pi.

Plug it in and turn it on – it's alive!

Meet the Terminal

Mac: press command space, type "terminal"

Windows: click the search icon, type "terminal"

It will look somewhat like this:

In the terminal:

% ssh frame@frame.lan
$ sudo raspi-config


Note! Replace "frame@frame.lan" with your username and hostname.

Raspberry Pi configuration:

Expand the filesystem:

Tap down to open "Advanced Options" then select "Expand Filesystem" from the menu.

Select "Finish" and reboot.

Back to the terminal!

% ssh frame@frame.lan
$ sudo apt update
$ sudo apt-get full-upgrade
$ sudo shutdown now


Note! Replace "frame@frame.lan" with your username and hostname.

Part 5: Software!

Again to the terminal:

% ssh frame@frame.lan
$ sudo apt-get install omxplayer
$ sudo apt-get install streamlink


Note! Replace "frame@frame.lan" with your username and hostname.

Are you creating a vertical frame?

$ sudo nano /boot/config.txt

config.txt

At the end of the file, add this:

display_hdmi_rotate=3

Once more, into raspi config

$ sudo raspi-config

Change screen resolution:

Tap down to open "Display Options" then select "Resolution" from the menu.

Choose DMT 85, then exit and reboot.
+ resolution settings screen

Let's try it!

streamlink --player-continuous-http --player omxplayer --fifo --player-args "--live --win \"0 0 1280 720\" {filename}" http://www.ustream.tv/channel/17074538 480p


Let's try it (vertical)!

streamlink --player-continuous-http --player omxplayer --fifo --player-args "--live --orientation 90 --win \"0 0 720 1280\" {filename}" http://www.ustream.tv/channel/17074538 480p

Let's make a script for this:

$ ls
$ touch iss-stream
$ ls
$ sudo nano iss-stream

Empty...for now

Copy this:

#!/bin/bash
while true
do

streamlink --player-continuous-http --player omxplayer --fifo --player-args "--live --orientation 90 --win \"0 0 720 1280\" {filename}" http://www.ustream.tv/channel/17074538 480p

done

+ screensnot of script in terminal

Make it executable:

$ sudo chmod u+x iss-stream
$ ls
$ ./iss-stream

Let's upload a video to the Pi:

% scp "path to file.mp4" frame@frame.lan:


Note! Run this command on your computer, not the Pi. Drag and drop files to the terminal.

+ scp terminal screenshot

Play & Loop video with Omxplayer:

$ omxplayer file.mp4
$ omxplayer --loop file.mp4

We can script this too!

#!/bin/bash
while true
do

omxplayer --loop ~/file.mp4

done


Note! You'll need to create the script file with "touch".

Let's upload a Pictures directory:

% scp -r "path to Pictures" frame@frame.lan:


Note! Run this command on your computer. We're sending files from your laptop to the pi.

+ scp -r terminal screenshot
